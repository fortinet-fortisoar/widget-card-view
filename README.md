## Overview

The **Card View** widget displays records from any FortiSOAR module as a visual grid of cards. Each card surfaces key record fields at a glance — including a header image, title, color-coded left border, and additional fields of your choice — making it easier to scan and assess multiple records without opening each one individually.

The widget was designed primarily for use with the **Threat Intel Management** solution pack, where it renders Threat Intel Reports, Cybersecurity News, and Threat Actors as visually rich cards. It can now be used independently with any FortiSOAR module.

### Key Features

- Display records from any module as a card grid

- Configure a header image, title, and left border color per card

- Add multiple custom fields to each card's body

- Filter records using single conditions or grouped condition sets

- Set a default sort order for the card grid

- Control the number of records displayed using a configurable limit
